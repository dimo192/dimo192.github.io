Simulation of α-ω Galactic Dynamo under slab approximation

The mean field dynamo equation

\(\frac{\partial \overline{\boldsymbol{B}}}{\partial t} = \nabla \times (\overline{\boldsymbol{V}} \times \overline{\boldsymbol{B}} + \mathcal{E} - \eta \nabla \times \overline{\boldsymbol{B}})\)

where, \(\mathcal{E} = \alpha \overline{\boldsymbol{B}} - \eta_{\mathrm{t}} \boldsymbol{\nabla} \times \overline{\boldsymbol{B}}\) and ignore the induction term \(\nabla \times (\overline{\boldsymbol{V}} \times \overline{\boldsymbol{B}})\). This can symplify the equation to

\(\frac{\partial \overline{\boldsymbol{B}}}{\partial t} = \nabla \times (\alpha \overline{\boldsymbol{B}} - (\eta_{\mathrm{t}}+\eta) \boldsymbol{\nabla} \times \overline{\boldsymbol{B}} )\)

Changing to cylinderical co-ordinates and assuming axis symmetric solution \(\frac{\partial}{\partial \phi}\)=0; and \(\frac{\partial}{\partial r}\)=0 due to slab approximation.

\(\frac{\partial \overline{\boldsymbol{B_r}}}{\partial \tilde{t}} = -\frac{\partial}{\partial \tilde{z}}(\tilde{\alpha}\overline{B_\phi})+\frac{\partial^2}{\partial z^2}(\overline{B_r})\)

\(\frac{\partial \overline{\boldsymbol{B_\phi}}}{\partial \tilde{t}} = -q\tilde{\Omega}\overline{B_r}+\frac{\partial^2}{\partial z^2}(\overline{B_\phi})\)

Converting all units to dimensionless units
  • \(\tilde{t}=\frac{t}{t_0}\)
  • \(\tilde{z}=\frac{z}{h_0}\)
  • \(\tilde{r}=\frac{r}{h_0}\)
  • \(\tilde{\alpha}=\frac{t_0\alpha}{h_0}\); where \(\alpha=\alpha_0\hat{\alpha}(\tilde{z})\) where \(\hat{\alpha}(\tilde{z})\) is an odd fucntion (eg. sin function)
  • \(\tilde{\Omega}=\Omega t_0\)
where, \(h_0\) is a unit of lenght; \(t_0\) is a unit of time with \(t_0=\frac{h_0^2}{\eta_t}\). Substituting this in above equation; we get

\(\frac{\partial \overline{\boldsymbol{B_r}}}{\partial \tilde{t}} = -\frac{\partial}{\partial \tilde{z}}(\tilde{\alpha}\overline{B_\phi})+\frac{\partial^2}{\partial z^2}(\overline{B_r})\)

\(\frac{\partial \overline{\boldsymbol{B_\phi}}}{\partial \tilde{t}} = -q\tilde{\Omega}\overline{B_r}+\frac{\partial^2}{\partial z^2}(\overline{B_\phi})\)

We can then define \(R_\alpha\equiv\frac{h\alpha_0}{\eta_t}\) and \(R_\Omega\equiv-\frac{h^2q\Omega}{\eta_t}\). Then critical dynamo number can be defined as

\(D=R_\alpha R_\Omega= -\frac{q\Omega \alpha_0 h^3}{\eta_T}\)

Calculation of critical dynamo number(\(D_c\))

The critical dynamo number is a number parameter that controls the regimes of the magnetic field amplification and decay in the system. The critical dynamo number is a dimensionless number that can be defined by

\(D=R_\alpha R_\Omega= -\frac{q\Omega \alpha_0 h^3}{\eta_T}\)

As the critical dynamo number can lead to the exponential growth or decay of a system it is important to calculate it for any system that is simulated.

The system is said to undergo exponential decay if dynamo number of system is less than the critical dynamo number (\(D< D_c \)) and exponential growth if dynamo number of system is more than the critical dynamo number (\(D> D_c \))

In the simulation the critical dynamo number is calculated in the following way

  1. Vary the dynamo number of the system.(This was done in the program by varying magnetic diffusion \(\eta_T\)).
  2. Calulate the variation of \(B_r\) and \(B_\phi\) with respect to time for this particular value of dynamo number.
  3. Calculate the norm at one particular z for every time considered.In my simulation z=0(as grid spans from z∈[-L,L]) was chosen.
  4. Then from t=\(t_0\); check if norm B at time t greater than norm B at time t'; where t> t' for any t'. This is done in a recurssion as the system may have reached a steady state at time t=\(t_0\).
  5. Store dynamo numbers for which the simulated system grows and take the minimum dynamo number. This is the crtical dynamo number.(In simulation \(\eta_T\) values were stored and maximum eta was taken to calclate minimum dynamo number.)
